## 常用chrome命令

mac版

##### 开发者工具

```
command + option + i
```

##### js控制台

```
command + option + j
```

##### 无痕新窗口

```
command + shift + n
```

##### 跳转标签页

```
command + 1~9
```

##### 关闭当前标签页

```
command + w
```

##### 打开历史记录

```
command + y
```

##### 地址栏焦点

```
command + l
```

##### 滚动一屏

```
空格
shift + 空格
```

##### 停止加载页面

```
Esc
```

##### 重置缩放

```
command + 0
```
